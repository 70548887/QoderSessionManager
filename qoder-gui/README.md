# Qoder 会话管理器

基于 Wails 框架的跨平台桌面应用，用于管理Qoder IDE的聊天会话。

## 功能特性

- 🖥️ **原生桌面应用** - 使用Wails构建，性能优秀
- 💾 **快速备份** - 一键备份所有工作区的聊天会话
- 🔄 **轻松恢复** - 快速恢复之前的会话记录
- 📊 **会话浏览** - 查看所有工作区的会话历史
- 📤 **导出功能** - 将会话导出为Markdown格式
- 🗑️ **备份管理** - 查看和删除旧备份

## 安装

应用已安装到 `/Applications/qoder-gui.app`

从启动台或应用文件夹双击启动即可。

## 使用方法

### 主界面布局

- **左侧面板**: 工作区列表（显示会话数量）
- **右侧面板**: 会话详情或备份列表
- **工具栏**: 备份、恢复、导出等操作按钮

### 典型使用流程

#### 切换用户前备份会话

1. 打开 Qoder 会话管理器
2. 点击 **"备份所有会话"** 按钮
3. 等待备份完成，状态栏会显示结果
4. 在Qoder中切换账号

#### 切换用户后恢复会话

1. 打开 Qoder 会话管理器
2. 点击 **"查看备份"** 按钮
3. 在备份列表中找到之前的备份
4. 点击 **"恢复"** 按钮
5. 确认恢复操作
6. 重启 Qoder，会话就恢复了

### 其他功能

- **导出会话**: 将所有会话导出为Markdown文本文件
- **删除备份**: 删除不需要的旧备份文件
- **刷新数据**: 重新加载工作区和会话数据

```bash
wails dev

# 构建生产版本
wails build
```

## 系统要求

- macOS 11.0+ (Big Sur) 或更高版本
- 支持 Intel 和 Apple Silicon (M1/M2/M3) 芯片

## 技术栈

- **后端**: Go 1.23 + Wails v2
- **前端**: Vanilla JavaScript + CSS
- **数据库**: SQLite3 (读取Qoder数据)
